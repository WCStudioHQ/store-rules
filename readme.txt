Store Rules offers role-based discounts in WooCommerce, allowing customized discounts for different user roles.

== Description ==

Store Rules is a powerful WordPress plugin designed to provide custom discounts based on user roles. Ideal for WooCommerce stores, it allows administrators to create tailored discount rules for specific customer groups, such as VIP members, wholesale buyers, or regular customers. By assigning unique discounts to different user roles, you can ensure that the right customers get the right prices.

Key Features:

* Apply unique discount percentages to different user roles (e.g., Admin, Customer, Wholesale).
* Automatically applies discounts at checkout, making the process smooth for your customers.
* Create, edit, or delete discount rules directly from the admin panel with a simple interface.
* Automatically prevents adding duplicate discounts for the same role to avoid conflicts.
* Discounts are applied automatically based on user role without manual intervention.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Click "WooCommerce -> Role Discounts" from  left menu.
4. From the dropdown select any role.
5. set any discount percentage as you want with decimal.
6. Finally click Updated Role and saved rules show in table .
7. For editing click "Edit" button then the rule discount show in the input field.
8. If you want to delete any rule click delete button.
